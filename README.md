# holojsony

Heavy overhaul fork of [jsony](https://github.com/treeform/jsony) for better usability in applications. I am not keen on licenses so if I am missing any credit anywhere for the fork I am ok with fixing it.

Currently about 2x slower, presumably due to the level of abstraction but I can't pinpoint why. (double pointer dereference for reading?) Still faster than the other libraries in the benchmarks though.

Not compatible with jsony's parsing/conversion behavior.
